# Design Reference

Mockups generated by Google Stitch, used as visual reference only.
Not shipped to production. Claude Code reads these to translate into
real Next.js + shadcn/ui + Tailwind components.

## Structure

design-reference/
└── stitch/
├── lumina_dark/
├── oncue_desktop_landing_page/
└── oncue_landing_page/

Each subfolder under `stitch/` contains the raw HTML, CSS, and any
image assets exported from a single Stitch generation. Folder names
correspond to the Stitch project/preset they came from.

- `lumina_dark/` — dark theme exploration
- `oncue_desktop_landing_page/` — desktop-focused landing page mockup
- `oncue_landing_page/` — responsive landing page mockup

When iterating on a new design, create a new subfolder rather than
overwriting an existing one — keeps the history of explorations
intact for reference.
